Questo progetto consiste nell'implementazione di un 
applicativo distribuito.
L'obittivo di questo progetto è quello di apprendere la 
metodologia DevOps e per capire come si applica nelle 
fasi di sviluppo di un software.

L'applicativo è un sito web sviluppato in WordPress il quale
si interfaccia con un DataBase configurato in MariaDB,
il quale viene utilizzato per la gestione dei dati.

L'ambienete di sviluppo è stato inserito all'interno di
containers e inoltre è stato implementata l'integrazione 
continua con travis.
